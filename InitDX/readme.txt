新增示例步骤:
1.新增一个win32空项目
2.设置项目依赖00comm项目
3.设置继承属性表..\lcm_d3d.vsprops
4.添加源文件.注意UNICODE编码

note1:从WinMain可以看出,每个例子都是win32类型的空项目.comm也是空项目,但是是静态库项目.
note2:使用项目属性继承的方式会让联合编译失败!

创建项目属性表步骤如下[摘自MSDN]:

1.在 Visual Studio IDE 中打开解决方案。
 
2.从“视图”菜单中选择“属性管理器”菜单项。“属性管理器”窗口将显示并停靠在 IDE 的左侧。
 
3.在“属性管理器”中，右击将包含新属性表的项目，然后选择“添加新项目属性表”菜单项。
 
4.在“添加新项”对话框中，在“名称”文本框中输入新属性表的名称。
 
5.单击“添加”。
